# Nokia Snake Game with Hand Gesture Control

A modern version of the classic Nokia Snake Game controlled using real-time hand gestures and computer vision.  
This project uses OpenCV, MediaPipe, and Pygame to create a touchless gaming experience.

---

# Features

- Classic Nokia-style Snake Game
- Real-time hand gesture control
- Swipe gestures for movement
- Pinch gesture for speed boost
- Hand tracking using MediaPipe
- Face detection support
- Particle effects when eating fruit
- Game restart using gesture
- Multi-threaded gesture processing
- Smooth gameplay with webcam integration

---

# Technologies Used

## Programming Language
- Python 3

## Libraries
- OpenCV → Webcam handling and image processing
- MediaPipe → Hand and face detection
- NumPy → Mathematical calculations
- Pygame → Game development and rendering
- Threading → Parallel gesture detection
- Time → FPS and timing control
